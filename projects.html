Pokemon Shuffle Move Analyzer

A simple move analyzer that gives you the best possible match based on current and future positions. Created for Pokemon Shuffle, a 3DS and Mobile game. Features save load functions. Written in Java with heavy use of JFrame for GUI.

Fishy!

Written in libGDX, Fishy! is a throw back to my highschool days, a reminisce of fish eat fish games. Eat smaller fish to get bigger. Fishy! features local high score saving, as well as working cross-platform through the use of the libGDX libraries.
